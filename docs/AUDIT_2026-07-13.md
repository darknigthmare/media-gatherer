# Audit complet - 2026-07-13

## Corrige

- Recherche trop permissive : score de pertinence centralise, modes strict/smart/broad et suppression des assets UI.
- Faux resultats sans le terme : les pages de moteur ne comptent plus comme preuve et les cartes doivent porter une preuve textuelle.
- Zero DuckDuckGo opaque : fallback HTML public et compteurs brut/filtre/pages ouvertes.
- Wayback incomplet : domaines via l'endpoint officiel, CDX jusqu'a 1 000 lignes, 714 images confirmees sur `sxysindy.com`, sans filtre de nom.
- NSFW generique : 22 adaptateurs declares avec domaines, chemins, types, limites de crawl et fallbacks publics.
- Mauvaise qualite : URL originale prioritaire, normalisation des variantes et dedoublonnage par media canonique.
- Videos sans miniature : poster reel ou placeholder, jamais une page HTML affichee comme image.
- Historique duplique : une ecriture consolidee par recherche complete au lieu d'une ligne par source.
- Queue fictive : worker synchrone borne avec etats, pause, reprise, annulation et progression.
- Persistance incoherente : magasin atomique local, statut explicite durable/temporaire et dossier externe au snapshot de l'EXE.
- Connexions trompeuses : tests reels Google/Brave/Flickr/YouTube et statut Telegram non implemente.
- API fragile : validation des IDs/URLs, 404 JSON, erreurs globales, CORS, CSP, rate limits et protection SSRF.
- UI dupliquee : quatre tabpanels exclusifs; Person Finder n'apparait plus dans les autres modules.
- Mobile : recherche et bouton empiles, navigation stable a 390 px, aucun debordement horizontal.
- Dependances CDN : Lucide, JSZip et FileSaver sont servis localement.
- Livraison : tests Node, CI GitHub, EXE Node 24 autonome et documentation Vercel.

## Verifications executees

- `npm run qa` : 9/9 tests.
- `npm audit` : 0 vulnerabilite.
- EXE : signature `MZ`, demarrage sans Node, `/api/health` disponible et stockage dans `dist/data/`.
- Recherche `sxysindy` sur DuckDuckGo+Bing : 35 images et 2 videos pertinentes.
- Wayback `sxysindy.com` : 829 lignes, 555 timestamps, 714 images, 5 assets UI retires.
- Navigateur : 1280x900 et 390x844, quatre onglets exclusifs et aucun overflow horizontal.

## Non cassable localement mais dependant d'un tiers

- Les sites avec login, paywall, challenge anti-bot, rendu JavaScript obligatoire ou restriction regionale peuvent refuser l'extraction.
- Les resultats Google, Brave, Flickr et YouTube dependent de quotas et d'identifiants API valides.
- Vercel ne fournit pas de disque durable pour ce magasin JSON.
- Telegram necessite une integration MTProto officielle et le consentement explicite du compte connecte.

Ces cas ne doivent pas etre presentes comme des succes : chaque source retourne un statut et une raison de zero ou d'erreur.
